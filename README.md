# Project Overview

The project simulates end-to-end data analyst workflows, from raw data exploration to business-focused analysis and insights.

# Key Objectives

Perform data cleaning and preprocessing to ensure dataset quality

Conduct exploratory data analysis (EDA) to identify patterns, trends, and anomalies

Generate actionable insights to support inventory and pricing decisions

Showcase SQL-based analytical skills and data visualization

# Skills & Tools

Programming & Analysis: SQL, Python (optional if you plan to visualize in Python)

Data Manipulation: Cleaning, filtering, aggregations, case statements

Visualization & Insights: Charts, summaries, top categories, pricing analysis

# Project Highlights

Cleaned dataset by handling nulls, zero prices, and inconsistent entries

Explored product categories, stock status, pricing, and discounts

Identified top 10 best-value products and categories with highest average discounts

Calculated estimated revenue and total inventory weight per category

Derived price-per-gram analysis and categorized products into Low, Medium, Bulk weight categories

# SQL Queries Used

Data exploration: SELECT COUNT(*), SELECT DISTINCT category, GROUP BY outOfStock

Data cleaning: DELETE, UPDATE statements

Analysis: Aggregations, CASE statements, revenue calculation, discount analysis
